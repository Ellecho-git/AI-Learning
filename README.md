## 🏗 笔记架构

本笔记采用模块化知识体系架构，按照学习阶段划分为以下几个核心模块：

- **导学篇** - AI 领域入门引导与专业名词讲解
- **基础篇** - Python 编程入门与数据科学工具包
- **应用篇** - AI Agent 智能体应用搭建
- **部署篇** - AI 大模型及开源组件环境部署
- **实战篇** - LangChain、Dify、RAG 等企业级应用实战
- **进阶篇** - L1 至 L5 各级别深度学习与项目实战
